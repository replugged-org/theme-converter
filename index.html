BetterDiscord → Replugged Theme Converter

Instructions

  1. Download a BetterDiscord theme (.theme.css).
  2. Upload it to the file picker below.
  3. Click "Convert" and allow the download if needed.
  4. Move the resulting .asar file to your Replugged themes folder.
  5. Click "Load missing themes" in Discord settings → Themes to apply the theme.
